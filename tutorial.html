推荐打开方式

最佳使用体验

建议使用 Microsoft Edge 浏览器（Windows 和 macOS 均可）打开本网站，语音识别效果最佳，操作更方便，输入更高效。Chrome 浏览器也可以正常使用。手机端也可以正常使用。

如何添加到电脑桌面（推荐）

Edge 浏览器（推荐）：打开网站后，点击地址栏右侧的"应用可用"图标（或菜单 → 应用 → 将此站点作为应用安装），选择"安装"
Chrome 浏览器：打开网站后，点击地址栏右侧的安装图标（或菜单 → 投放、安装应用）
安装后可以从桌面直接打开，像使用独立软件一样方便。

语音识别设置

在设置页面的「语音识别设置」区域，可以选择语音转文字引擎：

  • 浏览器内置：免费，使用浏览器内置语音识别（推荐 Edge 浏览器），需要联网
  • 本地AI识别：基于 OpenAI Whisper 模型，完全离线运行，无需联网，隐私安全。首次使用需下载模型文件（约40MB）
两种引擎对比

浏览器内置：实时识别，速度快，需要联网（Edge / Chrome 效果最佳）
本地AI识别：离线可用，隐私安全，首次需下载模型，识别速度取决于设备性能

使用建议

日常使用推荐「浏览器内置」，免费且速度快。
如果网络不稳定或无法使用浏览器语音识别，可切换到「本地AI识别」。
选择「本地AI识别」后，建议先在设置页面点击「预加载模型」提前下载，避免上课时等待。
无论使用哪种引擎，都可以直接在文本框中手动输入课堂内容，效果完全一样。

录音文件导入（语音转文字）

如果您已经用其他设备录好了课堂录音，可以使用录音文件导入功能将录音转为文字：

  1. 在录音页面，点击「录音」按钮下方的「导入录音文件」链接
  2. 选择您的录音文件（支持 mp3、wav、m4a、ogg、webm 等常见格式）
  3. 系统会自动播放录音并通过语音识别引擎将内容转为文字
  4. 转换完成后，文字会自动填入课堂内容文本框中
录音导入说明

使用「浏览器内置」引擎导入时，需要 Chrome 浏览器支持，转换速度约为录音时长的 1:1。
使用「本地AI识别」引擎导入时，完全离线处理，速度取决于设备性能（通常比实时更快）。
转换过程中可以正常使用其他功能，无需等待。

长时间录音支持

系统已针对长时间课程进行了专门优化，支持 2 小时一对一课程和 40 分钟试听课的稳定录音转文字。

长时间录音保障机制

自动重启：浏览器语音识别每隔约 20 秒会自动暂停，系统会在后台自动重启，全程无需手动操作。
健康检查：每 30 秒检查一次识别状态，如果发现识别卡住会自动恢复。
状态提示：录音过程中实时显示已录制时长和已识别字数。
最大时长：单次录音最长支持约 2 小时（360 次自动重启），到达上限后会自动停止并提示。
静音检测：如果 90 秒内没有检测到任何语音，会自动停止录音，避免浪费。

使用建议

上课时可以随时在文本框中补充或修改内容，录音转文字和手动输入可以混合使用。
如果语音识别不可用，直接在文本框中输入课堂内容即可，效果完全一样。

苹果电脑（macOS）使用说明

系统已针对 macOS 系统进行了适配优化，支持 Safari 和 Chrome 浏览器。

macOS 麦克风权限设置

步骤一：打开「系统设置」→「隐私与安全性」→「麦克风」
步骤二：确保您使用的浏览器（Chrome 或 Safari）已勾选
步骤三：刷新网页后重试

Safari 浏览器额外设置

如果使用 Safari 浏览器，还需要：
步骤一：Safari 菜单 → 偏好设置 → 网站 → 麦克风
步骤二：找到本网站，将麦克风权限设为「允许」
步骤三：确保 macOS 系统设置中也允许了 Safari 使用麦克风

推荐使用 Edge

在 macOS 上，Edge 浏览器的语音识别兼容性更好，推荐优先使用 Edge。如果 Safari 遇到语音识别服务不可用的提示，请切换到 Edge 重试。Chrome 浏览器也可以正常使用。

首次使用配置

首次打开网站后，需要完成以下三项基础配置：

  1. 点击左侧菜单或底部导航栏的「设置」图标
  2. 在「API Key」栏填入您的 DeepSeek API Key（获取方式见下方说明）
  3. 在「科目管理」中添加您教授的科目（系统已预置语文、数学、英语、政治、历史、地理、物理、化学、生物 9 科，可按需增删）
  4. 返回首页，点击右下角「+」按钮添加学生信息

如何获取 DeepSeek API Key

  1. 访问 platform.deepseek.com
  2. 注册账号并登录（支持手机号注册）
  3. 进入「API Keys」页面，点击「创建 API Key」
  4. 复制生成的 Key（以 sk- 开头），粘贴到设置页面中
费用说明

DeepSeek API 新用户有免费额度，后续费用约 ¥0.003/千字。单次 2 小时课程反馈约 ¥0.014，小组课（3人）约 ¥0.026。每月按每天 4 节课、22 天计算约 ¥1.65，充值 20 元可用半年以上。

日常使用流程

  1. 在「学生管理」页面找到要上课的学生，点击学生卡片
  2. 选择本次上课的科目
  3. 进入录音页面，点击「开始录音」按钮，课堂计时器自动开始计时
  4. 在文本框中输入本节课的课堂内容（如教学知识点、学生表现、作业安排等），也可使用快捷回复快速插入常用内容
  5. 点击「生成反馈」按钮，AI 将自动生成结构化课堂反馈（生成过程中会显示进度提示）
  6. 在弹出的反馈结果中，可以点击文字直接编辑修改
  7. 确认无误后，点击「一键复制」即可粘贴发送给家长

课堂计时器

录音页面顶部显示课堂时长计时器，录音开始时自动计时，停止录音时自动停止并记录时长。计时数据会保存在历史记录中，方便您追踪每节课的时长。

科目快捷切换

在录音页面，点击顶部的科目名称可以快速切换本次课程的科目，无需返回上一页重新选择。切换后不影响已输入的课堂内容。

小组课（一对多）

支持 1 对 2 或 1 对 3 的小组课模式：

  1. 在学生管理页面，点击「小组模式」按钮
  2. 依次点选参与上课的学生（最多 3 人）
  3. 点击底部出现的「确认选择」按钮
  4. 选择科目后，后续流程与单人课相同

小组模式姓名插入

在小组模式下，文本框上方会出现每位学生的姓名按钮（如「@小明」「@小红」）。点击按钮可在文本框中插入「@姓名：」标记，帮助 AI 区分不同学生的表现，生成更精准的个性化反馈。

使用建议

在记录课堂内容时，先点击姓名按钮插入标记，再输入该学生的表现。例如：「@小明：今天主动回答了3个问题，配方法掌握得很好」「@小红：二次函数图像理解有进步，但计算还需加强」。

小组模式反馈特点

小组模式下，AI 会为每位学生分别生成独立的反馈。「课堂内容」和「课后作业」模块为全组统一的公共内容（不出现学生姓名），「课堂表现」和「薄弱环节」等模块针对每位学生个性化生成。试听课学生的反馈标题会标注「试听」字样。

自定义反馈风格

在设置页面的「反馈风格」区域，您可以：

  • 选择 6 种语气风格：亲切友好、正式专业、简洁明了、详细具体、轻松幽默、积极鼓励
  • 设置每模块字数范围（最少/最多字数），默认为 50-150 字，课后作业默认 50-100 字
  • 开关「姓名截取」（三字名自动取后两字，如"王小明"→"小明"）
  • 开关「Emoji 表情」及位置（内容中/标题后/模块末尾）
  • 开关「请家长协助」内容
  • 开关「严格遵循输入内容」（防止 AI 编造未提及的内容）
  • 输入自定义要求（如"多使用鼓励性语言"等）
字数设置说明

全局字数限制为后备值，每个模块可单独设置字数范围。手动修改过的模块字数不受全局调整影响。超长课程（如 2 小时）建议将字数上限调至 1000-2000 字。

反馈模块管理

在设置页面的「反馈模块」区域，您可以自定义反馈的结构模块：

  • 启用/禁用模块：勾选或取消勾选模块前的复选框，控制生成反馈时包含哪些模块
  • 添加自定义模块：点击「+ 添加自定义模块」按钮，输入模块名称即可创建新的反馈模块，还可为自定义模块填写描述，帮助 AI 更精准地生成内容
  • 排序模块：通过模块右侧的「↑」「↓」按钮调整模块顺序，排序结果会影响生成反馈的模块排列顺序
默认模块

系统默认包含「课堂内容」「课堂表现」「薄弱环节」「课后作业」等模块。您可以根据需要添加如「家长建议」「学习计划」等自定义模块。

科目专属模板

在设置页面的「科目专属反馈模板」区域，可以为每个科目设置专属的生成要求。

示例

数学科目可填写："强调解题思路和公式推导过程，注重计算准确性"
英语科目可填写："关注单词记忆、语法运用和口语表达"

界面主题

在设置页面的「界面主题」区域，可以选择 4 种主题风格：

  • 默认紫：经典紫色主题，专业稳重
  • 温暖橙：暖色调主题，亲切温馨
  • 清新绿：绿色调主题，自然舒适
  • 深色：深色背景主题，适合夜间使用，保护眼睛
主题切换即时生效

选择主题后立即生效，无需刷新页面。主题偏好会自动保存，下次打开时自动应用。阴影和发光效果会跟随主题色变化。

快捷回复与常用点评

录音页面提供快捷回复功能，帮助您快速输入常用内容：

  • 在录音页面展开「快捷回复」区域，点击标签即可快速插入
  • 输入内容后，点击「保存为常用点评」可将当前内容保存
  • 每位学生有独立的常用点评库，方便复用

管理快捷回复

在设置页面的「快捷回复管理」区域，您可以添加、编辑和删除快捷回复：

  • 点击「+ 添加快捷回复」按钮，在底部弹窗中输入内容并选择分类（表扬/建议/作业/自定义）
  • 快捷回复按分类显示，方便快速查找
  • 鼠标悬停在标签上时，右上角会显示红色「×」按钮，点击可删除（支持撤销恢复）
  • 可以添加多个分类，如「课堂表现」「作业相关」「鼓励用语」等

历史反馈与科目筛选

在学生的历史反馈页面，可以查看该学生所有过往的反馈记录：

  • 历史反馈按日期自动分组展示，今天的反馈显示为"今天"，昨天的显示为"昨天"，更早的显示具体日期
  • 每条反馈卡片左侧有科目颜色标识，方便快速识别不同科目
  • 点击顶部科目筛选按钮，可以按科目查看反馈记录，切换学生时筛选自动重置
  • 点击任意反馈卡片可查看完整内容，并支持复制

生成学习总结

在学生的历史反馈页面，点击「生成学习总结」按钮，AI 会根据该学生近期的反馈记录，自动生成一份包含整体情况、主要进步、需要加强和后续建议的学习总结报告。

总结范围

系统会取最近 20 次反馈记录中的前 15 次进行分析，自动生成结构化总结报告，帮助您和家长全面了解学生的学习轨迹。

草稿自动保存

系统会自动保存您在录音页面输入的课堂内容草稿。如果您不小心关闭了页面或刷新了浏览器，再次打开时草稿会自动恢复。

草稿保存机制

每次输入内容时系统会自动保存草稿到本地。草稿与当前选中的学生和科目关联，切换学生或科目后会加载对应的草稿。生成反馈后草稿会自动清空。

存储空间监控

系统会实时监控本地存储空间使用情况。当存储空间使用超过 4MB（总容量约 5MB）时，会弹出橙色警告提示，建议您及时导出数据并清理旧记录。

存储说明

所有数据保存在浏览器本地（localStorage），不会上传到任何服务器。课堂时长记录最多保存 100 条，超出后自动覆盖最早的记录。

数据管理

所有数据保存在浏览器本地，不会上传到任何服务器。

重要提醒

清除浏览器数据会导致所有记录丢失！建议定期使用设置页面中的「导出数据」功能进行备份。导出的数据已自动移除 API Key，可安全分享。

  • 「导出数据」：将所有学生、反馈、设置导出为 JSON 文件备份（自动移除 API Key）
  • 「导入数据」：从备份文件恢复数据（更换浏览器或电脑时使用）
  • 「清空所有数据」：删除所有本地数据（操作前有确认弹窗，并支持撤销恢复）

常见问题

录音转文字无法使用？

请检查：1）是否使用 Edge 浏览器（推荐）或 Chrome；2）麦克风权限是否已允许；3）macOS 用户还需检查系统设置中的语音识别权限。如果仍无法使用，直接在文本框中输入课堂内容即可。

反馈生成失败？

请检查：1）API Key 是否正确填写；2）网络是否正常；3）DeepSeek 账户余额是否充足。系统对网络错误会自动重试一次，如仍失败会提示具体错误信息。

更换电脑后数据丢失？

数据保存在浏览器本地，更换电脑或浏览器后需要重新导入。请提前使用「导出数据」备份。

如何修改已生成的反馈？

反馈弹出后，直接点击文字内容即可编辑修改，修改后会自动保存。修改完成后再点击「一键复制」即可。

macOS 上语音识别不可用？

请检查：1）系统设置 → 隐私与安全性 → 麦克风，确保浏览器已勾选；2）Safari 还需在偏好设置 → 网站中允许麦克风；3）推荐使用 Edge 浏览器，兼容性更好。Chrome 浏览器也可以正常使用。

长时间录音中途断了怎么办？

系统会自动重启语音识别，正常情况下无需手动操作。如果提示"已达到最大录音时长"，说明已录制约 2 小时，这是单次录音上限。您可以停止后重新开始新的录音。

教程页面打不开或显示旧内容？

这是浏览器或 Service Worker 缓存了旧版本导致的。请尝试：1）强制刷新页面（Ctrl+F5 或 Cmd+Shift+R）；2）清除浏览器缓存后重新访问。最新版本已修复此问题。

复制反馈时提示"复制失败"？

这通常是因为浏览器不支持剪贴板 API（如非 HTTPS 环境）。系统会自动降级到传统复制方式，如果仍失败，请手动选中文本复制。
